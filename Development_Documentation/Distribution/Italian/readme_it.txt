===------------------------------------------------------------------------------===
I - Informazioni su Anathema
===------------------------------------------------------------------------------===

Anathema è un completo programma per gestire ogni aspetto della serie Exalted come
Personaggi e Campagne, Librerie Musicali, Note di Cronaca e molto altro.


===------------------------------------------------------------------------------===
II - Download e Installazione
===------------------------------------------------------------------------------===

Anathema è disponibile gratuitamente su: http://sourceforge.net/projects/anathema

Benchè il file compresso ZIP copre ogni esigenza, vi consigliamo di utilizzare
la specifica versione per il vostro Sistema Operativo, se disponibile.

Le guide all'installazione ed all'uso sono disponibili sulle sito del progetto.

===------------------------------------------------------------------------------===
III - Esecuzione
===------------------------------------------------------------------------------===

Eseguite il programma utilizzando la riga di comando "java -jar anathema.jar"
Gli utenti Windows possono utilizzare, alternativamente, il file eseguibile
Anathema.exe.
Su Mac OS X, il file DMG viene automaticamente integrato nel desktop.

Gli utenti Linux devono garantire le autorizzazione di lettura/scrittura sulle
cartelle oppure specificare quale cartella, con tali autorizzazioni, conterrà i dati
salvati dal programma, utilizzando il comando '-Drepository' es.:

java -Drepository="C:\AnathemaRepository" -jar anathema.jar

Questo memorizza i dati di Anathema nella cartella AnathemaRepository posizionata su C:

Anathema necessita della versione 5.0 di Java Runtime Environment(JRE).
Per gli utenti windows invece si ha bisogno di JRE 6.0.
Per determinare la versione di java installata digitare "java -version" dal prompt dei comandi

Si può scaricare l'ultima versione disponibile dal sito: http//java.sun.com

IV - Caratteristiche
===------------------------------------------------------------------------------===

+ Generare e memorizzare tutti e 5 i tipi di Eccelso
+ Generare una descrizione dei Personaggi(non disponibile attualmente)
+ Stampare la Scheda personaggio con descrizioni in PDF
+ Organizzare i Temi della cronaca
+ Gestire una raccolta musicale per la propria serie
+ Visualizzare i prodigi degli Eccelsi

Leggere il file versions.txt per lo storico dei miglioramenti di Anathema.

===------------------------------------------------------------------------------===
V - Personalizzazione
===------------------------------------------------------------------------------===

Per personalizzare Anathema, dal menù Extra avviare "Preferenze".
Tutte le modifiche fatte alle impostazioni, necessitano di un riavvio del programma
per avere effetto.

Una descrizione esauriente delle opzioni modificabili è disponibile nella
"Guida all'Installazione" presente sulla pagina del progetto

===------------------------------------------------------------------------------===
VI - Errori
===------------------------------------------------------------------------------===

A dispetto di quello che si può pensare, è molto probabile che Anathema non sia
esente da bug ed errori.

Se riscontrate qualsiasi problema, vi preghiamo di farcelo presente attraverso l'apposita pagina su
Sourceforge.

Grazie per la vostra collaborazione!


Problemi conosciuti:
---------------------
-"Pregi e Difetti" cambia dimensione quando si selezionano gli oggetti.

-E' stato riportato un problema quando si tenta si salvare l'equipaggiamento nella base dati, ma
non avviene in ogni condizione. Vi preghiamo di contattarci quando notate tale problema.

-"Avvio a schermo intero" non funziona correttamente su alcuni sistemi Linux.

Leggere il file versions.txt per l'elenco degli errori risolti fin'ora.

VIII - Contributi
===------------------------------------------------------------------------------===

Anathema è un progetto open source, quindi, si ha accesso gratuitamente anche
al codice sorgente.

Se hai un miglioramento da apporre, una grande idea per una nuova caratteristica o

l'integrazione.

Se sei interessanto a vedere Anathema nella tua lingua, per favore contattaci.

In questo momento, non accettiamo altre persone nel team di sviluppo. Le ragioni
di questo sono molteplici ma se ti senti comunque interessato inviaci una mail.
